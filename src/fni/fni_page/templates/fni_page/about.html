About FNI project

Inflation has been a global issue lately, though it is part of the economical cycle. The drastic increased in Consumer Price Index (CPI) has caused economical stress to the global citizens. This project aims to look into overall market sentiment by scrapping tweets from Twitter, and also the trend and topics of interest by looking into Google Trend. Lastly, these data are also compared to CPI data, to have a better understanding of how people react to inflation and their interest or curiousity surrounding the topic of inflation.

This is a study project done by Jiong Jiet and Iqram.

NLP python tools and Django web framework are the main stacks of this project.
